# Vila-real SportFest 2026

![Vista previa del proyecto](capturas/Captura.PNG)

Este es el repositorio del sitio web del Vila-real SportFest 2026, el evento deportivo que este año vuelve a las instalaciones municipales de Vila-real con más fuerza que nunca.

<br>

---

<br>

## Descripción del proyecto

He creado una página web informativa para el Vila-real SportFest 2026, un evento organizado por el Ayuntamiento de Vila-real que se celebrará los días 10, 11 y 12 de abril de 2026.

La web está pensada para que la gente pueda enterarse bien de qué va el evento, qué deportes hay, los horarios y cómo contactar con la organización. Me he centrado en que sea clara, que se vea bien tanto en el móvil como en el ordenador, y que tenga un diseño llamativo pero sin ser demasiado recargada.

Para maquetarla he usado Bootstrap 5, que me ha ayudado mucho a hacerla responsive sin tener que escribir demasiado CSS desde cero.

## Secciones y columnas de Bootstrap

Aquí explico cómo he organizado cada sección de la página y cómo he usado el sistema de columnas de Bootstrap en cada una:

**Navbar**  
Barra de navegación superior fija con fondo oscuro. Ocupa todo el ancho y se colapsa en móvil con el menú hamburguesa.

**Hero**  
Cabecera principal con imagen de fondo, título grande y dos botones. Ocupa todo el ancho.

**Qué es**  
Texto explicativo a la izquierda e imagen a la derecha.

**Modalidades**  
Una columna en móvil, dos en tablet y cuatro en escritorio para mostrar las tarjetas de los 4 deportes: fútbol 7, basket 3x3, carrera 5K, y pádel con fitness.

**Programa e info**  
Dividido en dos columnas:
- Columna izquierda: programa con lista de actividades.
- Columna derecha: información práctica con fechas, lugar y requisitos, más una foto.

**Contacto**  
Sección con fondo oscuro. El formulario va centrado.

**Footer**  
Dividido en dos partes: copyright a la izquierda y organización a la derecha.

## Componentes de Bootstrap que he usado

**Navbar**  
Para el menú de navegación, con menú hamburguesa en móvil y fondo oscuro.

**Sistema de columnas**  
Para maquetar todas las secciones.

**Tarjetas**  
En la sección de deportes, cada modalidad va dentro de una tarjeta con imagen, título y descripción.

**Botones**  
Para los llamados a la acción: botón azul de Inscríbete y botón outline en el hero.

**Listas**  
En el programa, para mostrar los horarios de viernes a domingo.

**Etiquetas**  
En el hero con la fecha del evento y en las tarjetas para indicar categorías y detalles.

**Utilidades**  
Clases para ajustar espacios y centrar texto.

## Historial de commits y mejoras

He ido subiendo los cambios poco a poco. También he creado una rama secundaria llamada **diseno-hero** y luego la he fusionado con pull request.

<br>

### Rama principal: main

**Initial commit**  
Primer commit con el archivo README.md creado.

**Add files via upload**  
Subida del archivo Pagina_WEB.html con toda la estructura de la página: navbar, secciones, tarjetas, programa, contacto y footer.

**Update README.md**  
Último commit subiendo este archivo con toda la documentación y las capturas.

<br>

### Rama secundaria: diseno-hero

**Add files via upload**  
En esta rama trabajé sobre el Pagina_WEB.html para mejorar el hero section:
- Ajusté la imagen de fondo.
- Cambié el alto del hero.
- Añadí la etiqueta con la fecha encima del título.
- Mejoré los botones.

<br>

### Fusión a main

**Merge pull request from diseno-hero**  
Cuando terminé los cambios en la rama secundaria, hice un pull request desde GitHub para fusionarlo con la rama principal.
